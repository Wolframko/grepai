## Why grepai?

Traditional search requires knowing exact names. grepai understands what you mean.

## Get started in seconds

```bash
# Install
curl -sSL https://raw.githubusercontent.com/yoanbernabeu/grepai/main/install.sh | sh

# Initialize
grepai init

# Index & Watch
grepai watch

# Search
grepai search "your query"
```

## grep vs grepai

| Traditional grep | grepai |
|-----------------|--------|
| Searches exact text | Searches by meaning |
| Returns 847+ noisy results | Returns top relevant matches |
| Requires knowing variable names | Works with natural language |
| `grep -r "auth" .` | `grepai search "user login flow"` |
